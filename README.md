## 🚀 Fix

- **Navbar** — Sticky navigation bar with smooth scroll links
- **Sliding Banner** — Image carousel with next/prev controls
- **Product Section** — Products fetched from [FakeStore API](https://fakestoreapi.com/products)
- **Shopping Cart** — Add/remove items, calculate total with delivery charge
- **Coupon System** — Use code `SMART10` for 10% discount
- **Balance System** — Start with 1000 BDT, add money dynamically
- **Search & Filter** — Filter products by name in real time
- **Reviews Section** — Customer testimonials
- **About Us** — Team info and company stats
- **Contact Form** — Name, email, and message form
